Description

This module contains PropEr's main parse transformer. It is automatically applied to modules when including the main PropEr header, unless PROPER_NO_TRANS is defined. Applying this transform has the following effects:
  • All prop_* functions of arity 0 in the module are automatically exported.
  • Type declarations in ?FORALLs that correspond to native types are properly substituted (with some limitations, see the README for details).
